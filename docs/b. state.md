## 1. STATE = “what exists right now”

A state is:
- the complete description of the system at a single moment

In your context:
- close
- mean
- median
- distance (derived state)

**Intuition to develop:**
- Think: “If I freeze time, what variables fully describe the system right now?”

**Key insight:**
- A state is not a feature — it is a snapshot of the system’s geometry
