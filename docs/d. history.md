## 3. HISTORY (MEMORY) = “what the system retains from past states”

Memory is:
- compressed information about past states that still influences current behavior

### In your system:
- rolling mean
- rolling scale
- n_above_mean / n_below_mean

### Intuition to develop:
- Think: “What part of the past still shapes how I interpret the present?”

### Key insight:
- Memory is not storage — it is compression of past into present bias
